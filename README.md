Data Generator
==========

A simple application to generate a specified number of personal data records.

Requirements
------------

Data Generator requires the following to run:

  * [Golang][golang] 1.10+


Usage
-----

Get the application:

```sh
go get github.com/martinroddam/datagen
```

Run the application:

```sh
datagen
```

Example output:

